# grandma-s-fun
A easy web extracting kit, by which even a grandma can enjoy the fun!

## Usage
#### set_props
set a k-v dict for self.props
    self.props = {
            "url": [],
            "next": ""
        }
key: prop name
value: default value, if extracting fail

####grap_propname
set a method the extract prop
